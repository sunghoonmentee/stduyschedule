# 📚 순공 플래너 (Sungong Planner)

예비 고1을 위한 **주간 공부계획표 + 순공시간 타이머** 웹앱.

## 기능
- 🗓️ **주간 계획표** — 요일별로 하고 싶은 과목을 적고 완료 체크
- ⏱️ **순공 타이머** — 뽀모도로식 타이머로 실제 공부 시간을 재서 자동 기록 (누적)
- 📅 **달력 기록** — 한 주가 끝나면 총 순공시간과 계획 완료율을 한눈에
- 💾 **백업 / 불러오기** — 기록을 파일로 저장하고 되살리기

## 사용법
`index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.
데이터는 브라우저에 자동 저장됩니다 (localStorage).

## 기술
- 순수 HTML / CSS / JavaScript (라이브러리 없음, 단일 파일)
