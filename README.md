# 📚 순공 플래너 (Sungong Planner)

예비 고1을 위한 **주간 공부계획표 + 순공시간 타이머** 웹앱.
설치형(PWA)이라 다른 컴퓨터·폰에서도 브라우저로 열어 "앱 설치"하면 오프라인에서도 쓸 수 있습니다.

## 기능
- 🗓️ **주간 계획표** — 요일별로 하고 싶은 과목을 적고 완료 체크
- ⏱️ **순공 타이머** — 스톱워치 / 시간 정하기(카운트다운) 2가지 모드, 잰 시간을 오늘 순공시간에 누적
- ☕ **휴식 타이머** — 분을 정해 카운트다운, 끝나면 소리 알림
- 📅 **달력 기록** — 하루·한 주의 순공시간과 계획 완료율을 정확한 시·분으로 표시
- 💾 **백업 / 불러오기** — 기록을 파일로 저장하고 되살리기 (기기 이동용)

## 바로 쓰기 (설치)
1. 공개 주소(GitHub Pages)로 접속: `https://sunghoonmentee.github.io/stduyschedule/`
2. 크롬/엣지 주소창 오른쪽의 **설치(⊕) 아이콘** → "설치" 클릭
3. 바탕화면/시작메뉴에 앱으로 추가됨 (오프라인에서도 작동)

> 로컬에서 쓸 땐 `index.html`을 브라우저로 열어도 됩니다. (단, "설치"와 오프라인 캐시는 https 주소에서만 동작)

## 데이터 보관
기록은 각 기기의 브라우저(localStorage)에 자동 저장됩니다. 다른 기기로 옮기려면 앱 하단의
**내보내기/가져오기**를 사용하세요. (기기 간 자동 동기화는 추후 클라우드 로그인으로 예정)

## 기술
- 순수 HTML / CSS / JavaScript (라이브러리 없음)
- PWA: `manifest.json` + 서비스워커(`sw.js`) + 아이콘
